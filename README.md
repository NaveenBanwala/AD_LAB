# AD Lab - Machine Learning Lab

This repository contains implementations of core Machine Learning algorithms and concepts completed as part of the AD Lab coursework.

## 📌 Topics Covered
- Introduction to Machine Learning
- Regression Analysis
- Multiple Linear Regression
- Logistic Regression (Classification)
- Decision Trees
- Random Forests
- Support Vector Machines (SVM)
- K-Means Clustering
- Principal Component Analysis (PCA)
- Neural Networks (TensorFlow/Keras)
- Convolutional Neural Networks (CNN)

## 🛠️ Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow / Keras

## 🎯 Purpose
This repository demonstrates practical implementation of machine learning algorithms for academic learning and skill development.

## 🚀 How to Run
- Clone the repository
- Install required libraries
- Run individual scripts/notebooks
